Help

This MVP demonstrates three required query workflows plus export and plot output.

  • Kinase query: Enter a UniProt accession (example: F4JWP9).
  • Condensate query: Enter a condensate UID (example: E12AB6E0).
  • Disease query: Enter a disease key (example: DOID:9119).
  • Each result page supports CSV export through the download link.
  • The Plot page generates and displays a PNG figure.
